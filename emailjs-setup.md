
# إعداد EmailJS لإرسال الإيميلات التلقائية

## الخطوات المطلوبة:

### 1. إنشاء حساب EmailJS
- اذهب إلى https://www.emailjs.com/
- أنشئ حساب جديد مجاناً

### 2. إعداد الخدمة (Service)
- في لوحة التحكم، اذهب إلى "Email Services"
- أضف خدمة Gmail وربطها بحساب Wareethofficial@gmail.com
- احفظ Service ID

### 3. إنشاء قالب الإيميل (Template)
- اذهب إلى "Email Templates"
- أنشئ قالب جديد بهذا المحتوى:

```
Subject: طلب انضمام جديد - فريق وريث

مرحباً،

تم استلام طلب انضمام جديد لفريق وريث:

الاسم الكامل: {{from_name}}
البريد الإلكتروني: {{from_email}}
رقم الهاتف: {{phone}}
الرسالة: {{message}}

تاريخ التقديم: {{application_date}}
وقت التقديم: {{application_time}}

يرجى مراجعة الطلب والتواصل مع المتقدم.

تحياتي،
نظام وريث الآلي
```

- احفظ Template ID

### 4. تحديث الكود
في ملف join.js، استبدل:
- `YOUR_PUBLIC_KEY` بـ Public Key من EmailJS
- `YOUR_SERVICE_ID` بـ Service ID من الخطوة 2
- `YOUR_TEMPLATE_ID` بـ Template ID من الخطوة 3

### 5. اختبار النظام
- املأ نموذج الانضمام واضغط إرسال
- تأكد من وصول الإيميل إلى Wareethofficial@gmail.com

## ملاحظات:
- الخدمة مجانية لـ 200 إيميل شهرياً
- يمكن تخصيص قالب الإيميل حسب الحاجة
- يتم الإرسال مباشرة من المتصفح بدون خادم
