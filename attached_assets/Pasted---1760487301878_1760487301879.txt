التغييرات التي قمت بها:
توحيد الخط العام: استخدمت font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; لجميع العناصر.

الحفاظ على الأيقونات: أضفت قاعدة CSS خاصة للأيقونات:

css
i, .fas, .far, .fab, .icon {
    font-family: 'Font Awesome 6 Free' !important;
}
هذه القاعدة تضمن أن الأيقونات تحافظ على خط Font Awesome الخاص بها.

استخدام الأيقونات الرسمية: استبدلت الرموز النصية بأيقونات Font Awesome الرسمية.

إضافة مكتبة Font Awesome: أضفت رابط CDN لمكتبة Font Awesome في رأس الصفحة.

بهذه الطريقة، أصبح الخط موحدًا في جميع النصوص مع الحفاظ على شكل الأيقونات كما هو دون تغيير.
