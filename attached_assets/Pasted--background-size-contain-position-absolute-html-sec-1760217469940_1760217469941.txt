التغييرات الرئيسية التي أصلحت المشكلة:

background-size: contain → cover - يمنع ظهور اللون الأخضر

إزالة position: absolute من النص - يجعل النص في المنتصف

إضافة background-attachment: fixed - تأثير parallax جميل

تحسين الشفافية - rgba(0, 0, 0, 0.3) بدل 0.2

إزالة top: -4% - كان يرفع النص لأعلى كثيراً

النتيجة النهائية:

✅ الصورة تظهر كاملة بدون أي ألوان خلفية

✅ النص في المنتصف تماماً

✅ متجاوب مع جميع الأجهزة
